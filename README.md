# FlashCards
HTML/CSS Document for generating Question/Answer cards for interviews

работает с любыми вопросами, хоть по 1С, хоть по сварке титана — если текст
соблюдает тот же “протокол карточек”, который парсер в HTML понимает.

___________________________________________________________________

Что именно должно быть в формате
Каждая карточка должна начинаться так:
N. <вопрос> (номер, точка, пробел, вопрос/заголовок)
дальше строки с метками (в любом порядке, но обычно так):

Ответ: ...

Пример: ... (необязательно)

Ошибка: ... (необязательно)


Пример минимальной карточки:

1. Что такое платформа 1С?

Ответ: ...

Пример полной:

2. Чем отличается “Конфигурация” от “Платформы”?

Ответ: ...

Пример: ...

Ошибка: ...

___________________________________________________________________


Что будет, если “вопросы по 1С” без такого формата?

Если там просто список вопросов без N.
или без Ответ: — HTML не сможет понять,
где заканчивается одна карточка и начинается другая, и всё “слипнется”.

Номер нужен только как разделитель карточек.
Языки/символы любые

___________________________________________________________________

В данный момент подготовленно для Java Junior Interview

cards.txt — исходник вопросов/ответов (Java backend)

cards_loader.html — генератор карточек (вопрос слева, ответ справа) | общий документ

Карточки.pdf — уже готовые карточки для печати | Java backend Junior Interview




Как пользоваться:

Скачай cards_loader.html

Открой cards_loader.html в Chrome/Edge

Выбери файл cards.txt на странице, можете заранее его подготовить для своих вопросов/ответов
(ВАЖНО по примеру моего cards.txt файла, иначе могут быть баги)

Нажми Print (Ctrl+P) → включи “Background graphics” если хочешь цвет
